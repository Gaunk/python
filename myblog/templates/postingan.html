Mendownload Python

Sebelum Anda mulai, Anda membutuhkan Python di komputer Anda. Periksa apakah Anda sudah menginstal versi terbaru Python dengan memasukkan python di jendela baris perintah. Jika Anda melihat respons dari interpreter Python, itu akan menyertakan nomor versi di tampilan awalnya. Umumnya versi Python 3.x apa pun bisa digunakan, karena Python berusaha keras untuk mempertahankan kompatibilitas ke belakang dalam versi Python utama. Python 2.x dan Python 3.x sengaja tidak kompatibel sepenuhnya. Jika python memulai interpreter Python 2.x, coba masukkan python3 dan lihat apakah versi terbaru sudah diinstal. Di Windows, coba py dulu - ini adalah Peluncur Python yang relatif baru, yang memiliki peluang lebih baik untuk menghindari beberapa masalah jalur yang mungkin terjadi karena pada program Windows tidak menginstal ke salah satu dari kumpulan kecil lokasi umum yang dicari secara default. Peluncur Python juga dapat membiarkan Anda memilih salah satu dari berbagai versi yang mungkin telah Anda instal dari satu perintah. Jika Anda perlu menginstal Python, Anda juga dapat mendownload versi stabil terbaru. Ini adalah yang memiliki angka tertinggi yang tidak ditandai sebagai rilis alfa atau beta. Silakan lihat halaman Unduh Python untuk versi terbaru dari Python. Mereka tersedia melalui tombol unduh kuning di halaman itu.

Windows

 Unduhan Windows paling stabil tersedia dari halaman Python untuk Windows . Di Windows, Anda memiliki pilihan antara versi 32-bit (berlabel x86 ) dan 64-bit (berlabel x86-64 ), dan beberapa jenis penginstal untuk masing-masingnya. Tim inti Python berpikir harus ada default yang tidak perlu Anda pikirkan lagi, jadi tombol unduh berwarna kuning di halaman unduh utama memberi Anda pilihan "x86 executable installer". Ini sebenarnya pilihan yang bagus: Anda tidak memerlukan versi 64-bit meskipun Anda memiliki Windows 64-bit, Python 32-bit akan bekerja dengan baik. Jika Anda menjalankan Windows XP : panduan lengkap untuk menginstal ActivePython ada di Python di XP: 7 Menit Untuk "Hello World!" . ShowMeDo memiliki dua video untuk mengunduh, menginstal, dan memulai Python pada mesin Windows XP - seri ini berbicara kepada Anda melalui distribusi Python, ActivePython , dan SciPy . Perhatikan bahwa rilis python.org hanya mendukung versi Windows yang didukung oleh Microsoft (pada saat rilis), jadi tidak ada rilis terbaru dari python.org yang dapat digunakan di Windows XP.

Mac

Lihat Python untuk Mac OS X. MacOS dari 10.2 (Jaguar) hingga 10.15 (Catalina) menyertakan versi sistem Python 2, tetapi sebaiknya jangan menganggap ini sebagai Python untuk digunakan untuk tugas pemrograman Anda - instal versi Python 3.x saat ini sebagai gantinya. MacOS setelah 10.15 (Catalina) tidak akan menyertakan sistem default Python.

Linux

Untuk Red Hat, CentOS atau Fedora , instal paket python3 dan python3-devel. Untuk Debian atau Ubuntu , instal paket python3.x dan python3.x-dev. Untuk Gentoo , instal ebuild '= python-3.x *' (Anda mungkin harus membuka kedoknya terlebih dahulu). Untuk sistem lain , atau jika Anda ingin menginstal dari sumber , lihat halaman unduhan umum .

Dokumentasi Python

Baca Dulu Dokumentasinya :

Python adalah bahasa pemrograman yang kuat dan mudah dipelajari. Ini memiliki struktur data tingkat tinggi yang efisien dan pendekatan yang sederhana namun efektif untuk pemrograman berorientasi objek. Sintaks yang elegan dan pengetikan dinamis Python, bersama dengan sifatnya yang ditafsirkan, menjadikannya bahasa yang ideal untuk pembuatan skrip dan pengembangan aplikasi yang cepat di banyak area pada sebagian besar platform. Penerjemah Python dan pustaka standar yang luas tersedia secara gratis dalam bentuk sumber atau biner untuk semua platform utama dari situs Web Python, https://www.python.org/ , dan dapat didistribusikan secara bebas.
Situs yang sama juga berisi distribusi dan penunjuk ke banyak modul Python pihak ketiga gratis, program dan alat, serta dokumentasi tambahan.
Penerjemah Python mudah diperluas dengan fungsi dan tipe data baru yang diimplementasikan dalam C atau C ++ (atau bahasa lain yang dapat dipanggil dari C). Python juga cocok sebagai bahasa ekstensi untuk aplikasi yang dapat disesuaikan. Tutorial ini memperkenalkan pembaca secara informal ke konsep dasar dan fitur dari bahasa dan sistem Python. Memiliki interpreter Python yang berguna untuk pengalaman langsung, tetapi semua contoh adalah mandiri, sehingga tutorial dapat dibaca secara off-line juga. Untuk deskripsi objek dan modul standar, lihat The Python Standard Library . Referensi Bahasa Python memberikan definisi bahasa yang lebih formal. Untuk menulis ekstensi di C atau C ++, membaca Memperluas dan Embedding Python Interpreter dan Python / C API Reference Manual . Ada juga beberapa buku yang membahas Python secara mendalam. Tutorial ini tidak berusaha untuk menjadi komprehensif dan mencakup setiap fitur, atau bahkan setiap fitur yang umum digunakan. Alih-alih, ini memperkenalkan banyak fitur Python yang paling penting, dan akan memberi Anda gambaran yang bagus tentang rasa dan gaya bahasa. Setelah membacanya, Anda akan dapat membaca dan menulis modul dan program Python, dan Anda akan siap untuk mempelajari lebih lanjut tentang berbagai modul pustaka Python yang dijelaskan di The Python Standard Library .
